## 🚀 What I've Been Building

### 🕸️ [Dynamic Voting Patterns](https://github.com/CharanKumarKharade/eurovision-viz)
*Academic Research for GD 2026 Competition*
I built a multivariate network analysis model to visualize how Eurovision voting shifts over time. It involved transforming massive temporal datasets into something you can actually see and understand using Python and NetworkX.

### 🚕 [Uber Real-Time Trip Analytics](https://github.com/CharanKumarKharade/real-time-trip-analytics)
*Azure Data Lakehouse*
An end-to-end simulation of Uber's surge pricing. I used **Azure Event Hubs** and **PySpark Streaming** to process real-time data into a Medallion-style Delta Lake, cutting down latency for pricing insights.

### 📈 [E-Commerce Performance suite](https://github.com/CharanKumarKharade/e-commerce-sales-dashboard)
*Power BI & Star Schema*
Analyzed 10k+ transactions. The focus here was data modeling—I used Star Schema and Z-Ordering to ensure the dashboards didn't just look good, but performed instantly under load.
